# Talibinden - CodeIgniter Migration & Design Update

## Proje Hakkında

Bu projede, geçen dönem geliştirdiğimiz **Talibinden** adlı web sitesini **CodeIgniter** altyapısına taşıyacak ve tasarımını güncelleyeceğiz. Proje, **CodeIgniter**, **HTML**, **CSS**, **MySQL** ve **MongoDB** teknolojilerini kullanarak daha modern ve kullanıcı dostu bir platform geliştirmeyi amaçlamaktadır.

## Kullanılan Teknolojiler

- **CodeIgniter**: PHP tabanlı bir MVC (Model-View-Controller) framework'ü, projede backend geliştirmede kullanılacak.
- **HTML5**: Web sayfasının yapısını oluşturmak için kullanılacak.
- **CSS3**: Web sayfasının tasarımını düzenlemek için kullanılacak.
- **MySQL**: İlişkisel veritabanı yönetimi için kullanılacak.
- **MongoDB**: NoSQL veritabanı, esnek veri yönetimi için kullanılacak.

## Proje Özellikleri

- **MVC Yapısı**: Proje, **CodeIgniter** framework'ü kullanılarak **Model-View-Controller** (MVC) mimarisi ile geliştirilecektir.
- **Responsive Tasarım**: Web tasarımı, farklı cihazlarda düzgün çalışacak şekilde güncellenerek **mobil uyumlu** hale getirilecektir.
- **Veritabanı Entegrasyonu**: Projede **MySQL** ve **MongoDB** veritabanları kullanılacaktır. **MySQL** ilişkisel veriler için, **MongoDB** ise daha esnek veri yapıları için kullanılacak.
- **Kullanıcı Yönetim Sistemi**: Kullanıcı girişi, kayıt işlemleri ve yönetim paneli gibi özellikler eklenecektir.
